# JAYYXXY Premium Twitch Streamer Website

A modern, premium gaming portfolio website designed for Twitch streamer **JAYYXXY**. Inspired by top esports organizations and professional gaming brands, this project combines a futuristic mascot-style identity with immersive animations, cinematic visuals, and a clean user experience.

The website serves as a central hub where fans, friends, and supporters can discover the streamer, explore featured games, access social media, and instantly join live Twitch streams.

## Features

- 🎮 Premium esports-inspired UI
- 👤 Custom mascot branding
- ⚡ Smooth animations and transitions
- 💙 Blue & White futuristic theme
- 🖼️ Gaming artwork and immersive backgrounds
- 📅 Streaming schedule
- 🎥 Featured gameplay section
- 🏆 Gaming statistics
- 🌐 Social media integration
- 🔴 Direct "Watch Live on Twitch" CTA
- 📱 Fully responsive design
- 🚀 Portfolio-ready modern interface
